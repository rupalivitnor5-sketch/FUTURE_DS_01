Project: Business Sales Dashboard & Insights

Description:
This project analyzes business sales data to generate key KPIs and insights. It includes Total Sales, Total Orders, Average Order Value, Profit, and visualizations for regional and category performance.

Tools Used:
- Microsoft Excel
- Power BI (optional)

Files:
- Clean_Data.xlsx       → Cleaned dataset
- Dashboard.xlsx        → Dashboard with KPIs & charts
- Insight_Report.xlsx   → Optional insights in Excel
- Dashboard.png         → Optional screenshot
- Insight_Report.pdf    → Optional PDF summary
- ReadMe.txt            → Project description

Key KPIs:
- Total Sales =₹ 21,57,978.71
- Total Orders = 8,840
- Average Order Value (AOV) = ₹244.12
- Total Profit = ₹ 2,61,277.01
- Profit Margin % = [12.11%]

Insights:
- East region contributes 30% of total sales but profit margin is low.
- Top-selling products contribute 45% of revenue.
- West region sales steadily increasing last 3 months.

Recommendations:
- Focus marketing on high-margin products in East region.
- Run discount campaigns on low-performing categories.
- Monitor monthly AOV to improve customer order value.
